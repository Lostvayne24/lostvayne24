Hobi

Hobi saya adalah bermain game. Sejak masih duduk di bangku kelas 2 SD, saya sudah gemar bermain game baik game offline maupun game online. Saat itu saya sering pergi ke Warung Internet (Warnet) bersama teman-teman setelah pulang sekolah, berbagai game pernah saya mainkan, seperti Dragon Nest, Lost Saga, Point Blank dan Digimon Master Online. Ketika saya kecil, bermain game hanya sebatas ajang melepas penat dan hiburan semata. Berbeda ketika saya sudah beranjak remaja, tepatnya ketika SMA Kelas 11. Saat itu sedang ada Pandemi Covid-19, saya yang seharusnya ada di tanah rantau terpaksa harus pulang ke rumah guna melepas mata rantai penyeberan virus.

Singkat Cerita, saya memiliki waktu luang yang banyak ketika di rumah dan saya gunakan untuk bermain game. Ternyata, game yang saya mainkan dapat menghasilkan uang. Sejak saat itulah orientasi saya dalam bermain game mulai berubah, yang sebelumnya hanya sebatas mengisi waktu luang sekarang sudah memikirkan bagaimana caranya dapat menghasilkan uang dari bermain game dan bisa lebih produktif.

Sampai saat ini, ketika memiliki waktu luang saya gunakan untuk bermain game dan berusaha tetap produktif dengan menghasilkan uang dari hasil bermain game.
